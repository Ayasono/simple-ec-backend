# KINS backend API

## 1. Introduction

The project is a backend API for the KINS project. It is a RESTful API that provides endpoints for the KINS frontend to
interact with the backend.

## 2. used technologies

- gin-gonic
- sqlc
- docker
- postgres
- JWT

## 3. implemented endpoints

- /users
  - get /
  - get /:email
  - post /
- /products
  - get /
  - get /categories
